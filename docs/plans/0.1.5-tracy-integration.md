## 3. Overview & Approach

### 3.1 Strategy in one paragraph
Add a plain `tracy` to the pinned manifest; add a `cmake/profiling.cmake` module that always defines the `aero::profiling` INTERFACE library (include dir → `engine/core/include`) and, only under `AERO_ENABLE_PROFILING`, links `Tracy::TracyClient` and defines `AERO_PROFILING_ENABLED`; write the backend-agnostic wrapper header; flip the option ON in the three `*-release` presets; link `aero::profiling` into `aero_tests` and add a test that exercises the wrappers in both configs. Verify Debug (no-op) + Release (Tracy) locally on the Mac; the PR's CI run proves all 3 OSes.

### 3.2 Load-bearing sequencing & correctness constraints
Treat each as load-bearing, never cleanup:
1. **`include(cmake/profiling.cmake)` placement.** After `project()` (compiler known) and after the `option(AERO_ENABLE_PROFILING …)` line, and **before any `add_subdirectory()`** — the interface target must exist before `add_subdirectory(tests)` links it. Correct home: immediately after the `include(cmake/sanitizers.cmake)` line, before `enable_testing()`.
2. **The interface lib is defined in BOTH states.** `add_library(aero_profiling INTERFACE)` + the include dir are unconditional; only `find_package`/link/define are inside `if(AERO_ENABLE_PROFILING)`. If it were defined only when ON, `target_link_libraries(aero_tests … aero::profiling)` would error in Debug.
3. **`find_package(Tracy)` is inside the ON branch only.** A default/Debug build must not require the Tracy package (criterion 6). vcpkg still *installs* tracy (manifest mode) but it is not found/linked when OFF.
4. **`AERO_PROFILING_ENABLED` (ours), not `TRACY_ENABLE` (Tracy's), drives the header** — keeps the header backend-agnostic (D6). `TRACY_ENABLE` still arrives transitively from `Tracy::TracyClient` so Tracy's own headers activate.
5. **No `TRACY_NO_EXIT` in CI.** It is a *manual*-only env var for observing a short-lived run; setting it in CI would hang `ctest` (E4).
6. **Do not enable any vcpkg `tracy` feature.** `gui-tools`/`cli-tools` pull glfw3/freetype/curl/capstone/imgui (E3). Plain `"tracy"` = client only.
